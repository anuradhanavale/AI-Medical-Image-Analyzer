# AI Medical Image Analyzer

An AI-powered medical image analysis application built using Streamlit and Google Gemini 2.5 Flash. The app analyzes uploaded medical images and generates structured AI-based diagnostic reports.

## Features

- Medical image upload and analysis
- AI-generated structured diagnostic reports
- Non-medical image detection
- Interactive Streamlit-based UI
- Secure API key handling using Streamlit Secrets

## Tech Stack

- Python
- Streamlit
- Google Gemini API
